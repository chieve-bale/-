结构力学求解器cumt
##单位统一为：长度mm，力N，应力MPa，默认E=206000MPa ，角度默认 °（逆时针为正）

力的类型有待商榷
#力的类型暂定为
  0：节点力
  1：节间力
    0：集中水平力
    1：集中竖向力
    2：斜向力
    3：集中剪力
    4：集中力偶
  5：均布力，需要补充长度（分数表示）、起点长度和终点长度[均布力距离起点的长度，均布力距离终点的长度]和端点值[起点值，终点值]

##杆件索引使用杆件的编码不使用节点编码！！！
